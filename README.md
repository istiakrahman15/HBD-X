# 🎌 Anime Birthday Cinematic Website

A cinematic anime-style interactive birthday surprise website built using **HTML, CSS, and JavaScript (Vanilla)**.

Inspired by anime opening styles like **Naruto, Attack on Titan, and Bleach**, this project delivers a smooth emotional storytelling experience with animations, music, and glowing UI effects.

---

## 🌟 Features

- 🎬 Cinematic anime-style intro scene  
- ✍️ Typing animation for title  
- 🌌 Interactive night sky background  
- 💖 Heartbeat animation on name  
- 🎞️ Scroll-based story chapters  
- 📜 Memory timeline (2011 → present)  
- 💬 Animated quote slider system  
- 🎂 Candle cake & birthday reveal section  
- 📸 Photo gallery (anime polaroid style)  
- 🎵 Background music support (music.mp3)  
- ✨ Floating stars, hearts & sakura effects  
- 📱 Fully responsive design (mobile + desktop)
